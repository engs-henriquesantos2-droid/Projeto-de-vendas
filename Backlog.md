# 📋 Backlog do Produto - Sistema de Vendas
**Responsável (PO):** [ Henrique Dos Santos ]

Este documento contém o planejamento de todas as funcionalidades do sistema, priorizadas para o desenvolvimento.

## 🔴 Prioridade Alta (Essenciais)
1. **Definição de Requisitos:** Levantamento de funcionalidades com o cliente.
2. **Modelagem do Banco de Dados:** Desenhar tabelas de produtos e vendas.
3. **Configuração do Repositório:** Estrutura inicial do projeto no GitHub.
4. **Interface de Login:** Tela de acesso para vendedores e administradores.
5. **Cadastro de Produtos:** Formulário para nome, preço e código.
6. **Listagem de Estoque:** Tabela visual com todos os itens cadastrados.
7. **Fluxo de PDV (Venda):** Seleção de itens para o carrinho de compras.
8. **Cálculo de Total:** Soma automática de itens com impostos.
9. **Finalização de Venda:** Botão para concluir e dar baixa no estoque.
10. **Segurança de Acesso:** Proteção de dados sensíveis do sistema.

## 🟡 Prioridade Média (Importantes)
11. **Cadastro de Clientes:** Registro de compradores fiéis.
12. **Histórico de Pedidos:** Tela para consultar vendas do dia/mês.
13. **Aplicação de Descontos:** Campo para reduzir valor total da venda.
14. **Edição de Preços:** Atualização de valores de itens já cadastrados.
15. **Gestão de Categorias:** Organizar produtos por tipo (ex: Alimentos, Higiene).
16. **Relatório de Sangria:** Controle de retiradas de dinheiro do caixa.
17. **Emissão de Comprovante:** Gerar versão para impressão do recibo.
18. **Barra de Busca:** Pesquisa rápida de produtos por nome.
19. **Alerta de Estoque Crítico:** Notificação quando um item está acabando.
20. **Backup em Nuvem:** Cópia de segurança automática do banco de dados.

## 🟢 Prioridade Baixa (Desejáveis)
21. **Cadastro de Fornecedores:** Contatos de quem fornece os produtos.
22. **Dashboard de Lucros:** Gráfico visual de faturamento mensal.
23. **Módulo de Trocas:** Sistema para devolução de mercadorias.
24. **Multi-pagamentos:** Opções para Pix, Cartão e Dinheiro separadamente.
25. **Importação via Excel:** Cadastro em massa de novos produtos.
26. **Exportação de PDF:** Gerar relatórios mensais para contabilidade.
27. **Perfil de Funcionário:** Troca de senha e foto do usuário.
28. **Log de Alterações:** Rastro de quem mudou preços no sistema.
29. **Interface Customizável:** Opção de Modo Escuro/Claro.
30. **Manual do Sistema:** Guia rápido de uso para novos funcionários.
